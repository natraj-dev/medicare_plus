# 🏥 MediCare Plus - Hospital & Appointment Management Platform

MediCare Plus is a full-stack Hospital Management System designed to streamline healthcare operations by providing appointment booking, patient management, doctor scheduling, prescriptions, billing, medical records, emergency services, and more.

## 🚀 Features

### 👨‍⚕️ Doctor Module

* Manage appointments
* View patient records
* Create consultations
* Write prescriptions
* Manage weekly schedules
* Slot management
* Lab test management

### 🧑‍🤝‍🧑 Patient Module

* Book appointments
* View prescriptions
* Manage medical records
* Track lab tests
* Health tracker
* Insurance management
* Emergency requests
* SOS contacts

### 👨‍💼 Admin Module

* User management
* Department management
* Doctor management
* Appointment monitoring
* Billing & payments
* Emergency management
* Reports & analytics
* Audit logs

## 🛠️ Tech Stack
